# Number row characters:
- \~ - tilde
- \! - exclamation mark
- \@ - at symbol  
- \# - hashtag  
- \$ - dollar sign
- \% - percent  
- \^ - caret
- \& - ampersand
- \* - star
- \( - left bracket
- \) - right bracket
